Project Manager

Please note - files in the project manager do not follow the same layout as your file system.

Adding/Removing Files

To add a file, right click a folder on the project manager and select 'Add File'. Browse to the file you want to add, then click OK.

To remove a file or folder, right-click it and select a subitem from the 'Delete' menu.

Renaming Files and Folders

Right-click and select 'Rename'. If you choose to rename a file, it is also renamed on the PC's file system.
