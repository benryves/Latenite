Tools

About

The tools menu is a simple way to add a bunch of useful shortcuts to Latenite. Environment variables are sent in a similar way to build scripts - so you can create tool scripts that, for example, open the current source file in Notepad.

What's allowed in the Tools folder?

To add or edit a tool, take a look in the Tools folder. The following sorts of files are allowed:

  • EXE/COM/CMD/BAT files - these are executed with some environment variables set. The variables are the same as the ones used for build scripts with the exception of ERROR_LOG and any variables starting with BUILD_*.
  • Any other files - these are run as if they were double-clicked on in Explorer.

One useful thing to note; placing an exclamation mark (!) as the first character of the tool's filename will allow it to open a command prompt window.
